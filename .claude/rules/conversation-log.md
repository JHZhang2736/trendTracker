# 对话历史记录

将所有对话历史记录在 `docs/conversation_history.md` 文件中。
- 用户的提问必须记录
- AI 的回复可以省略或简写
- 此文件已从 git 跟踪中移除（在 .gitignore 中），仅保留在本地
- 重要的架构决策应记录在 GitHub Issue 讨论中，而非对话记录
